
目前比较流行的数据库：

1.MS-SQL     -> 微软    -> C#     -> 大型
2.MySql      -> 甲骨文  -> JAVA   -> 中小型   -> 免费、开源代码、完全开放、跨平台
3.Oracle     -> 甲骨文  -> JAVA   -> 超大型   -> 海量数据存储
4.DB2        -> IBM     -> JAVA   -> 海量     -> 海量数据存储

以上全部为关系型数据库，

数据库的发展方向：对象型数据库。

几种数据库的内置超级管理员账号：
SQL Server ------ sa
MySql -------- root
Oracle -------- sys 或 system
