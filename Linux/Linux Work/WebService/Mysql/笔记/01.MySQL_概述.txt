MySQL
--------------------------------------------------------------------------------------------------------
SQL 不区分大小写，包括字段名。

mysql数据库管理系统安装
开始-》运行-》cmd

开始mysql服务：
C:\>net start mysql
MySQL 服务正在启动 .
MySQL 服务已经启动成功。

停止关闭mysql服务：
C:\>net stop mysql
MySQL 服务正在停止.
MySQL 服务已成功停止。

连接MySql服务：
第一种方式：（可用于远程）
C:\Program Files\MySQL\MySQL Server 5.0\bin>mysql -hlocalhost -uroot Cp（后边不要；号，要的话错！！）
Enter password:
Welcome to the MySQL monitor.  Commands end with ; or \g.
Your MySQL connection id is 2
Server version: 5.0.41-community-nt MySQL Community Edition (GPL)

Type 'help;' or '\h' for help. Type '\c' to clear the buffer.
（将localhost换成IP地址就可以连接远程，如：mysql Ch202.102.23.1 Cuadmin Cp）
第二种方式：（本机）
C:\Program Files\MySQL\MySQL Server 5.0\bin>mysql -uroot -p
如果没有设置密码还可以以这种方式：
C:\Program Files\MySQL\MySQL Server 5.0\bin>mysql -uroot

退出MySql服务：
mysql> exit
Bye


移除mysql服务：
先停止服务：
C:\Program Files\MySQL\MySQL Server 5.0\bin>net stop mysql
MySQL 服务正在停止.................................
MySQL 服务已成功停止。
再移除：
C:\Program Files\MySQL\MySQL Server 5.0\bin>mysqld --remove mysql
Service successfully removed. 

安装服务：
C:\Program Files\MySQL\MySQL Server 5.0\bin>mysqld --install mysql
Service successfully installed.
再重新启动服务：
C:\Program Files\MySQL\MySQL Server 5.0\bin>net start mysql
mysql 服务正在启动 .
mysql 服务已经启动成功。


使用解压版MySql：
1．解压后放到一个目录下，如：c:\mysql;
2．再在控制台下运行如下命令：
C:\>cd mysql

C:\mysql>cd bin

C:\mysql\bin>mysqld --install mysql
Service successfully installed.

C:\mysql\bin>net start mysql
mysql 服务正在启动 ..
mysql 服务已经启动成功。

一些常用的SQL语句：
--查看字符编码方式
show variables like 'char%'

--设置编码方式
set character_set_database=gbk

--查看MySql的版本
select version();

--查询所有mysql账户信息
select host,user,password from mysql.user;

--设置root账户的密码
set password=password(‘123’);
set password=password(‘’);

--将sql语句的结束符更换成//
Delimiter  // 

--显示所用数据库
show databases; 

--显示所有的表
show tables;

--执行某段SQL代码源文件
source c:\aa.sql(后边不能要分号，要的话就错了！！！)

--查看mysql支持哪几种数据库引擎
show engines;

--查看mysql数据库连接数、并发数
show status like 'Threads%';
+-------------------+-------+
| Variable_name     | Value |
+-------------------+-------+
| Threads_cached    | 16    |
| Threads_connected | 38    |   ###这个数值指的是打开的连接数
| Threads_created   | 54    |
| Threads_running   | 12    |   ###这个数值指的是激活的连接数
+-------------------+-------+

--查询数据库当前设置的最大连接数
show variables like '%max_connections%';

--显示当前正在执行的mysql连接
show processlist;
